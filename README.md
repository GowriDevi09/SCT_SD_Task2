# 🎯 Number Guessing Game

## 📌 Project Overview

This project is developed as part of the **SkillCraft Technology Software Development Internship (Task 2)**.

The Number Guessing Game is a simple and interactive web application where the computer generates a random number between **1 and 100**, and the player has to guess it. The game provides hints after each guess and keeps track of the number of attempts.

---

## 🚀 Features

- 🎲 Random number generation (1–100)
- 📈 Displays "Too High" hint
- 📉 Displays "Too Low" hint
- 🎉 Displays success message on correct guess
- 🔢 Tracks the number of attempts
- 🔄 New Game button to restart the game
- 📱 Responsive and user-friendly interface
